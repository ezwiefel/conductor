# Conductor Implement Skill

This skill executes tasks defined in a track's plan.

## 1. Initialization
First, read the system directive and setup check.
- [System Directive](references/01_system.md)
- [Setup Check](references/02_setup_check.md)

## 2. Track Selection
Identify and select the track to be implemented.
- [Track Selection Protocol](references/03_track_selection.md)

## 3. Implementation
Execute the selected track's plan.
- [Implementation Protocol](references/04_implementation.md)

## 4. Documentation
Synchronize project documentation with the completed track.
- [Synchronization Protocol](references/05_synchronization.md)

## 5. Cleanup
Archive or delete the completed track.
- [Cleanup Protocol](references/06_cleanup.md)
