# Conductor Revert Skill

This skill assists in reverting work by analyzing git history and implementation plans.

## 1. Initialization
First, read the system directive and setup check.
- [System Directive](references/01_system.md)
- [Setup Check](references/02_setup_check.md)

## 2. Selection
Guide the user to select the track, phase, or task to revert.
- [Selection Protocol](references/03_selection.md)

## 3. Verification
Verify the git history and identify commits to revert.
- [Verification Protocol](references/04_verification.md)

## 4. Confirmation
Present the plan and get user confirmation.
- [Confirmation Protocol](references/05_confirmation.md)

## 5. Execution
Execute the reverts and verify the plan state.
- [Execution Protocol](references/06_execution.md)
